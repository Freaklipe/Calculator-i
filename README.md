## Calculadora/Interoperabilidad
### Descripción
Este proyecto es una calculadora muy sencilla, que está diseñada con tres módulos/proyectos en vb, C# y F#. La cual solo realiza funciones de aritmética básicas.
### Finalidad
Es ejemplificar la interoperabilidad de los leguajes de programación bajo el framework .Net
